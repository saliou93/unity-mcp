
### Workflows avancés
Le système propose des templates structurés pour :
- Configuration rapide de prototypes (15 minutes)

### Intégration MCP
Compatible avec tous les outils Unity MCP :
- `manage_script`, `manage_gameobject`, `manage_scene`

- Validation automatique et bonnes pratiques Unity

## Note technique
Ce repository `unity-mcp` est un pont MCP (Model Context Protocol) qui permet aux assistants IA d'interagir avec Unity Editor. Il contient déjà des systèmes sophistiqués de prompts pour l'édition de code Unity et les tests automatisés.
